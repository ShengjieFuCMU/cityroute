# Glossary
- **POI (Point of Interest)**: A place the user wants to visit (museum, park, landmark).
- **Day Plan**: The set and order of POIs for a given day.
- **Detour**: Extra travel time added to include a restaurant/hotel.
- **Open Hours**: Lunch/dinner windows when a restaurant is open.
- **Lock**: A fixed choice that must be kept when regenerating the plan.
- **Export**: Download the plan to PDF/CSV/JSON.
